# Poke Team Photo

## Usage

* Install dependencies
```sh
npm install
```

* Run locally (debug)
```sh
npm start
```

* deploy to github-pages
```sh
npm run deploy
```

## Sources used for fetching sprites

* Pokemon Sprites - https://pokeapi.co/
* Pokemon Sprites - https://github.com/PokeAPI/sprites (e.g. https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/versions/generation-i/yellow/1.png)
* Trainer Sprites - https://play.pokemonshowdown.com/sprites/trainers/?view=dir
